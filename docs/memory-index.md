# 项目记忆索引

项目：RE:Flip（翻转重启）
阶段：1（原型冲刺）
核心玩法：分裂棋盘俄罗斯方块，方块从中间出生，翻转选方向，消行传送对面
最后巩固：设计短会完成，里程碑确认

## 有效决策

- D-001 ★★★ 棋盘结构：10列×30行，中线在15-16行之间，上半区15行+下半区15行
- D-002 ★★★ 出生位置：方块从中线出生（不是从边缘）
- D-003 ★★★ 翻转按钮：选择方块去上半区（向天花板堆积）或下半区（向地板堆积）
- D-004 ★★★ 消行判定：上下两半独立判定满行
- D-005 ★★★ 消行重启：消行后相邻外侧行传送到对面，从中间自动下落到底
- D-006 ★★☆ 传送行控制（打磨阶段）：M-003 改为可左右移动、不可旋转
- D-007 ★★★ Game Over：中间出生点被堵住，新方块无法放入
- D-008 ★★★ 方块种类：经典7种(I/O/T/S/Z/L/J) + SRS旋转
- D-009 ★★★ 操控：虚拟按键(左/右/旋转/翻转/硬降) + PC键盘
- D-010 ★★☆ 视觉风格：极简霓虹，深色背景，竖屏
- D-011 ★★☆ 已放置方块不受翻转影响
- D-012 ★★☆ 节奏参数：先用标准俄罗斯方块默认值
