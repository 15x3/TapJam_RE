# 游戏设计简报

## 必填

### 核心动词

点击翻转重力（决定方块落向哪一端）

### 胜负条件

- 得分：消除行数累计
- 失败：上下两摞方块在中间碰头（Game Over）

### 一句话描述

重力翻转版俄罗斯方块——点一下，方块飞向另一端，消除的行会"重启"到对面，永恒循环直到崩溃。

## 选填

### 屏幕方向

竖屏（手机优先，PC 自适应）

### 视觉风格

极简几何 + 霓虹配色，深色背景

### 参考游戏

俄罗斯方块（网格消行）+ 重力翻转机制

### 目标玩家

Jam 评委 + 休闲玩家

### 关键体验

"管理两个战场"的紧张感 + 消行重启到对面的循环宿命感

### 必须有的功能

- 单一点击交互
- 消行重启机制
- 分数显示
- Game Over 重玩

### 绝对不要的东西

- 不要复杂的方块旋转（单格方块即可）
- 不要多种操作方式
