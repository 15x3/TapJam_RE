# 游戏设计简报

## 必填

### 核心动词

移动/旋转经典俄罗斯方块，翻转重力方向，消行触发对面重启

### 胜负条件

- 得分：消行 × 等级 × 连击倍率
- 失败：新方块无法放入网格（Game Over）

### 一句话描述

经典俄罗斯方块 + 重力翻转 + 消行重启：消除整行后，一个全新方块从对面重新出现，玩家必须同时管理上下两端。

## 选填

### 屏幕方向

竖屏（手机优先，PC 自适应）

### 视觉风格

极简几何 + 霓虹配色，深色背景

### 参考游戏

俄罗斯方块（经典7种方块 + 旋转 + 消行） + 重力翻转机制

### 目标玩家

Jam 评委 + 休闲玩家

### 关键体验

"管理两个战场"的紧张感 + 消行重启到对面的循环宿命感

### 必须有的功能

- 经典 7 种俄罗斯方块（I/O/T/S/Z/L/J）+ 旋转
- 虚拟按键操控：左移、右移、旋转、翻转重力、硬降
- 翻转重力按钮（改变所有方块落下方向）
- 消行重启机制（消行后 1 个随机新方块从对面中央出现）
- 重启方块有短暂调整时间后自动硬降
- 出生位置居中，被堵 = Game Over
- 逐渐加速的紧凑节奏
- 分数 / 等级 / 连击显示
- PC 键盘控制 + 手机虚拟按键

### 绝对不要的东西

- 不要单格方块（必须经典方块组合）
- 不要滑动操作（使用虚拟按键）
